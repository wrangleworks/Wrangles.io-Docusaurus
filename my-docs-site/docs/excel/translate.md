Our Translation Wrangle is powered by DeepL.

[Sign up for your own DeepL API key](https://www.deepl.com/pro#developer) and then enter your key into the Translation Task Pane settings. A free account for up to 500,000 characters per month is available.


> Click [here](/python/recipes/wrangles/standalone#translate) to learn how to use Translate Wrangles in a recipe.

# Usage

To translate data, simply select your data, choose a language to translate from and a language to translate to in the drop-down menu then click Translate.
