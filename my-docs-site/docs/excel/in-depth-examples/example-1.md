Lets assume the following product data needs to be reformatted into a product description column for import into your target system:

![demo_-_products.png](/images/demo_-_products.png)

- Brands
- Prices
- Countries
- Shapes

- Materials

- Items
- Random Words
- Weights
- Lengths
- Storage
- Curly Brackets
- Uppercase
- 0000 At The Beginning Of Codes
- Storage Size
- Fractions
- White Space
