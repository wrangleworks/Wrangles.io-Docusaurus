# Using Extract Stock Wrangles

Extract stock wrangles are pretrained models that either 1)find and retrieves a predefined list of words within text or 2) uses pattern recognition AI to find the words. This technology enables us to extract keywords from data.

To use Stock Wrangles, press the **Data Wrangles** tab, above the ribbon. Once here, a new tool ribbon will appear. Click **Extract**. This will open the Wrangles Task Pane where Stock Wrangles can be used.
